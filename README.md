# Mini Blog Website
A lightweight, minimal, and easy-to-deploy mini-blog website. 
Built for simplicity, speed, and readability. Ideal for beginners 
learning web development or anyone who wants a quick personal blog without heavy frameworks.

4.Lightweight HTML, CSS, and JavaScript
5.Beginner-friendly project structure
